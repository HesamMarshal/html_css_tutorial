General

  • Copy: Ctrl - C
  • Cut: Ctrl - X
  • Paste: Ctrl - V
  • Save: Ctrl - S
  • Search: Ctrl - F

Vs Code

  • Hide Activity Bar: Ctrl - B
  • Split Screen: Ctrl - \
  • Close file: Ctrl - W
  • Command Palette: Ctrl-shift-p
  • Go to File: Ctrl - P
  • Terminal: Ctrl-`
  • Duplicat a Line: Alt-Shift-(Down)
  • Move a Line Up: Alt-(Up)
  • Move a Line Down: Alt-(Down)

Google Chrome

  • View Page Source: Ctrl-U
  • Developer Tools: Ctrl-Shift-I
  • Page Reload: Ctrl-R
  • Page Reload(Clear cache): Shift-F5

Live Server

  • Run Curret File: Alt-L Alt-O
