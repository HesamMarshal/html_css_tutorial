General

  • Undo: Ctrl - Z
  • Redo: Ctrl - U

  • Copy: Ctrl - C
  • Cut: Ctrl - X
  • Paste: Ctrl - V

  • Save a file: Ctrl - S
  • Open a file: Ctrl - O

  • Search: Ctrl - F
  • Replace: Ctrl - H

Vs Code

  • Comment a line: Ctrl - /
  • Hide Activity Bar: Ctrl - B
  • Split Screen: Ctrl - \
  • Close file: Ctrl - W
  • Command Palette: Ctrl-shift-p
  • Go to File: Ctrl - P
  • Terminal: Ctrl-`
  • Duplicat a Line: Alt-Shift-(Down)
  • Move a Line Up: Alt-(Up)
  • Move a Line Down: Alt-(Down)
  • Go to line: Ctrl - G
  • Intellisense: Ctrl - Space
  • New File: Ctrl - Win - Alt - N
  • Move between open files:
    Ctrl - P Ctrl -p
  • Search Element (current file):
    Ctrl - P @
  • Select two item to edit: alt
  • Select next Item: Ctrl -D
  • Select all items : Ctrl - F2

Google Chrome

  • View Page Source: Ctrl - U
  • Developer Tools: Ctrl - Shift - I
  • Page Reload: Ctrl - R
  • Page Reload(Clear cache):
    Shift-F5
